PVP형 엔드 컨텐츠 : 콜로세움, 아레나
PVE형 엔드 컨텐츠 : 협동전, 카마존, 부유성 미궁, 경계의 미궁, 궤도 엘리베이터

// PVP형
콜로세움 : 비동기 PVP, 유저의 개입은 공덱과 방덱, 위치 설정뿐 -> 보상 젬

아레나 : 실시간 PVP, 전투가 가능한 시간대를 설정하여 보다 빠르고 안정적인 매칭을 유도 함, 상위 랭크로 올라갈수록 과금도 같이 필요한 구조
           또한, 시즌 초기화 마다 맵이 변경되어 맵에 따른 사용 영웅이 달라짐 -> 보상 젬


// PVE형
협동전 : 가장 최근에 추가된 컨텐츠로 최대 4인의 유저가 하나의 맵을 클리어 하는 형태 (타게임의 레이드와 비슷한듯)
           협동전도 매칭이 필요하기 때문에 입장 가능 시간을 제한하여, 빠르고 안정적인 매칭을 유도 -> 보상 젬

카마존 랜드 : 지난주에 갈아 엎어져서 좀 어려운데..

부유성 미궁 : 달리기, 퍼즐, 일반 전투, 보스 전투 총 4가지로 구성되어 있으며 캐릭터의 성장과 플레이어의 컨트롤 성장을 기대 할 수 있음.
	     -> 보상은 캐릭터 성장에 관련된 물품들 강화용 망치, 경험치, 골드, 히어로 크리스탈 등등

경계의 미궁 : 부유성 미궁을 베이스로 난이도가 더욱 강화된 형태, 속성 제한이 생겨 속성 덱에 대한 활용도를 넓힘
	     -> 보상은 인게임 재화들, 코스튬
